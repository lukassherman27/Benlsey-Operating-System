# Research Inbox

Drop files here with AI tips, articles, tool recommendations, etc.

## How to use:
1. Save article/screenshot/notes as a file here
2. Name it descriptively: `2026-01-02-mcp-servers-list.md`
3. Run: `/research-inbox` (or ask Claude to process inbox)

## Claude will:
- Read all files in this folder
- Extract actionable insights
- Create GitHub issues for improvements
- Update the system based on findings
- Move processed files to `processed/` subfolder

## Example content to add:
- Twitter threads about AI coding
- Blog posts about MCP servers
- YouTube video notes
- Tool recommendations
- Workflow tips from other developers
