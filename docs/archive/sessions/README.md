# Session Notes

Development session summaries and status updates.

## Naming Convention

- `SESSION_SUMMARY_YYYY-MM-DD.md` - Daily session notes
- `CURRENT_STATUS.md` - Latest project status
- `TODAYS_ACHIEVEMENTS.md` - Daily progress

## Purpose

These documents help:
- Track what was accomplished each session
- Maintain context between Claude conversations
- Document decisions made during development

## Cleanup

Session notes older than 30 days can be archived or deleted.
